required sibling of the SPI/I2S/TDM HAL).

Upstream repository:

- Repository: https://github.com/sulaolab/dspic33ak-hal-dma
- Branch: main
- Source directory: src/

Synchronized into this repository under:

- Destination directory: src/hal_dma/
